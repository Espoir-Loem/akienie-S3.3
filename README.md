# Landing Page - CSS Avance

Projet S3.3 de l'Academie Akieni. Landing page moderne avec CSS avance.

## Description

Landing page style produit tech integrant les concepts CSS avances :
specificite, selecteurs combinateurs, pseudo-classes, variables CSS, cascade et heritage.

## Sections

1. Hero - Grand titre avec image de fond
2. Fonctionnalites - 3 colonnes avec icones
3. Testimonials - Avis clients
4. Pricing - 3 plans compares
5. CTA - Appel a l'action
6. Footer - Pied de page

## Comment utiliser

```bash
# Cloner le depot
git clone https://github.com/Espoir-Loem/akienie-S3.3.git

# Acceder au dossier du projet
cd akienie-S3.3

# Ouvrir la page dans le navigateur
start landing/index.html
```

Ou ouvrez simplement le fichier `landing/index.html` dans votre navigateur.

## Concepts CSS mis en oeuvre

- Specificite maitrisee : selecteurs cibles sans `!important`
- Selecteurs combinateurs : `+` (frere adjacent), `~` (frere general), `>` (enfant direct)
- Pseudo-classes : `:nth-child`, `:first-of-type`
- Variables CSS : `--primary-color`, `--secondary-color`
- Cascade et heritage : `inherit`, `initial`
- Reset CSS personnalise

## Criteres remplis

- Pas de conflit CSS
- Variables CSS utilisees
- Code organise et commente
- Design professionnel et coherent
- Minimum 5 sections differentes
- Selecteurs CSS3 avances
- Accessible (contraste adequat)
